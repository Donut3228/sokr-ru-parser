# sokr.ru parser

* [Install](#Install)
* [Usage](#Usage)

## Install

```bash
cd /opt
git clone https://github.com/Donut3228/sokr-ru-parser.git
cd sokr-ru-parser
python3 -m venv venv
python3 -m pip install -r requirements.txt --no-cache-dir
```

## Usage

```bash
python3 sokr_ru.py crawler run
python3 sokr_ru.py parser parse
```
